# Linear regression from scratch

This directory contains an implementation of a multiple linear regression.


# Quick start

Install dependencies:

```
$ make install
```

Run tests:

```
$ make test
```

# Dataset

Petrol dataset: https://people.sc.fsu.edu/~jburkardt/datasets/regression/x14.txt
